#  CEH-QCM

**A lightweight web application designed to help you train for the Certified Ethical Hacker (CEH) certification.**

This app provides an interactive Multiple Choice Question (MCQ) interface to test your knowledge of cybersecurity concepts and hacking techniques.

---

###  Features

* **Question Bank:** Initially curated by [@traygeur](https://github.com/Traygueur).
* **Engine:** Built using the **Flask** framework (Python).
* **Development:** Web interface and logic designed with the assistance of AI.

---

###  Getting Started

Follow these steps to get your local development environment up and running:

1. **Clone the repository**
```bash
git clone https://github.com/your-username/ceh-qcm.git
cd ceh-qcm

```


2. **Install requirements**
The application requires Flask. You can install it using pip:
```bash
pip install flask

```


3. **Run the application**
Start the local development server:
```bash
python app.py

```


4. **Access the app**
Open your favorite browser and navigate to: `http://127.0.0.1:5000`

---

###  Technical Requirements

* **Python 3.x**
* **Flask**
